The Price is Right Come on Down!

Come on down! Avec 4 jeux différents sur le même billet, voici votre chance de gagner au jeu The Price is Right Come on Down! Trouvez des symboles identiques pour gagner des lots et courir la chance de jouer à des jeux bonis. Dans le jeu Cliff Hangers, pourrez-vous faire grimper le tyrolien jusqu’en haut de la falaise? Ou trouverez-vous un multiplicateur en tournant la roue Showcase Showdown? Tentez votre chance à votre jeu télévisé préféré!

Comment miser

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (-) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter le prix du billet de votre solde et commencer la partie.

Comment jouer

Le but du jeu The Price is Right Come on Down! est de découvrir tous les symboles du jeu principal. Le jeu principal est composé de quatre rangées jouables : deux rangées dans le Jeu 1 et deux rangées dans le Jeu 2.

Jeu 1 : Découvrez les trois symboles de chaque rangée. Gagnez le lot lorsque tous les symboles sont identiques.

Jeu 2 : Découvrez tous les symboles de chaque rangée. Gagnez le lot lorsque l’un de vos symboles est identique à un symbole Gagnant de la même rangée.

Les symboles Boni peuvent apparaître dans le jeu principal. Lorsque trois symboles sont révélés, le compteur Cliff Hangers est complet, et le jeu boni Cliff Hangers commence à la fin du jeu principal. Si vous découvrez le symbole Showcase Showdown, le jeu boni Showcase Showdown commence à la fin du jeu principal.

Dans le jeu boni Cliff Hangers, appuyez sur le bouton JOUER pour activer le cylindre. Lorsque le cylindre s’immobilise, le tyrolien monte autant de marches sur la montagne que le nombre affiché sur le cylindre. Si le tyrolien s’arrête dans un secteur affichant un lot, vous gagnez ce lot. Le cylindre continue de tourner jusqu’à ce que le tyrolien s’arrête en haut de la falaise ou qu’il en tombe. Tous les lots amassés durant la montée sont accordés. Vous pouvez arrêter le tyrolien durant sa montée en appuyant sur ARRÊTER.

Dans le jeu boni Showcase Showdown, la grande roue est remplie par les multiplicateurs suivants : 2, 3, 5, 10, 25 et 100. Lancez la roue. Le gain total est multiplié par le multiplicateur indiqué lorsque la roue s’immobilise.

Tout afficher

Appuyez sur TOUT AFFICHER pour afficher rapidement tous les résultats du jeu principal. Si des jeux bonis sont déclenchés après que le joueur a appuyé sur TOUT AFFICHER, ils sont joués normalement.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les lots instantanés sont ajoutés aux gains totaux.

Les gains des jeux bonis sont additionnés aux gains du jeu principal.

Tous les gains du jeu principal et des jeux bonis sont multipliés par les multiplicateurs du jeu Showcase Showdown.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
